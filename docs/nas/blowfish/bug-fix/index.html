2. 文章封面#

發現 Blowfish 預設無法單獨設定文章縮圖，於是把 Cover 關鍵字拿來當文章縮圖。文章圖片的關鍵字有四個並分為兩種：

  • cover/thumbnail/featured: 縮圖+進入文章後的照片
  • background: 只有進入後的照片

把 layouts/partials/hero 中所有的 *cover*, 刪掉即可。
